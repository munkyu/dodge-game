# 팩맨 스타일 게임

HTML5 Canvas와 JavaScript로 만든 간단한 팩맨 스타일의 게임입니다.

## 게임 설명
- 방향키나 터치 컨트롤을 사용하여 캐릭터를 조작합니다
- 빌런들을 피해 최대한 오래 생존하세요
- 모바일과 데스크톱 모두 지원합니다

## 스테이지
1. 소정이 게임
2. 엄마 게임

## 조작 방법
- PC: 방향키로 이동, 스페이스바로 시작/재시작
- 모바일: 화면 하단의 방향키로 이동, 화면 터치로 시작/재시작

## 플레이 방법
[여기를 클릭하여 게임 시작하기](https://munkyu.github.io/my-pacman/)

## 기술 스택
- HTML5 Canvas
- JavaScript
- CSS3
